# Tema Zonaweb

Tema Zonaweb (`zonaweb_website`) is a complete Odoo 18 module that transforms your Odoo instance into a professional corporate website for IT services companies. It provides a polished blue/navy brand experience with purpose-built pages, a bidding request system, and a security plan pricing section.

## Brand colors

The theme ships with three CSS custom properties declared in the layout template:

```css
:root {
  --primary:   #0057FF;   /* Main blue — buttons, links, highlights */
  --secondary: #0B1F3A;   /* Dark navy — hero backgrounds, section fills */
  --accent:    #00C2FF;   /* Cyan — labels, badges, hover states */
}
```

## Module dependencies

The module extends the standard Odoo website stack:

- **`website`** — Core Odoo website module (layout, pages, menus)
- **`website_sale`** — Odoo e-commerce (shop links for plan purchases)
- **`mail`** — Messaging and email template support
- **`sale_management`** — Sales workflow integration
