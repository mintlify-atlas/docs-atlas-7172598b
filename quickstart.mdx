# Quickstart

Get Tema Zonaweb running on your Odoo 18 instance in under 10 minutes.

## Prerequisites

- Odoo 18.0 instance (Community or Enterprise)
- Administrator access to the Odoo backend
- The following Odoo apps already installed: **Website**, **eCommerce**, **Discuss** (`mail`), **Sales**

## Installation

<Steps>
  <Step title="Copy the module to your addons path">
    Clone or copy the `zonaweb_website` directory into any folder listed in your Odoo `addons_path`.

    ```bash
    # Example: placing it alongside other custom modules
    cp -r zonaweb_website /opt/odoo/custom-addons/
    ```

    Then confirm your `odoo.conf` includes this path:

    ```ini odoo.conf
    addons_path = /opt/odoo/odoo/addons,/opt/odoo/custom-addons
    ```
  </Step>

  <Step title="Restart the Odoo server">
    Restart the Odoo service so it picks up the new module:

    ```bash
    sudo systemctl restart odoo
    # or, if running manually:
    python odoo-bin -c /etc/odoo/odoo.conf
    ```
  </Step>

  <Step title="Update the apps list">
    In the Odoo backend, go to **Settings → Activate developer mode**, then navigate to **Apps → Update Apps List**. Confirm when prompted.
  </Step>

  <Step title="Install the module">
    Search for **Tema Zonaweb** in the Apps list and click **Install**.

    <Note>
      Odoo will automatically install the required dependencies (`website`, `website_sale`, `mail`, `sale_management`) if they are not already active.
    </Note>
  </Step>

  <Step title="Visit your website">
    Navigate to **Website** in the top menu, then click **Go to Website**. You should now see the Zonaweb corporate homepage with hero carousel, services section, and portfolio preview.
  </Step>
</Steps>

## Verify the installation

After installation, these URLs should all respond with styled pages:

| URL | Page |
|---|---|
| `/` | Homepage with hero, services, portfolio, pricing, testimonials |
| `/servicios` | Full services catalog |
| `/portafolio` | Filterable project portfolio |
| `/nosotros` | About / team page |
| `/licencias` | Bidding request form |

## Submit your first bidding request

To verify the bidding system works end-to-end:

<Steps>
  <Step title="Open the bidding form">
    Go to `/licencias` on your website. You will see a form asking for company name, contact details, service type, project description, budget, and timeline.
  </Step>

  <Step title="Fill in the form and submit">
    Complete all required fields and click **Submit**. The form posts to `/submit-bidding` with CSRF protection enabled.
  </Step>

  <Step title="Check the confirmation page">
    You should be redirected to a thank-you page showing your bidding reference number (e.g., `BID/00001`).
  </Step>

  <Step title="Review the record in the backend">
    In the Odoo backend, navigate to **Website → Bidding Requests** (or search for `zonaweb.bidding.request` in debug mode) to see the new record in `draft` state.
  </Step>
</Steps>
