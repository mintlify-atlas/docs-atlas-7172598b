## Page layout

Each service uses an alternating row layout defined by `.zw-service-row`:
- **Odd rows (1, 3):** image left, text right — standard `.zw-service-row`
- **Even rows (2, 4):** text left, image right — `.zw-service-row.zw-service-row--reverse` (achieved via `direction: rtl` on the row)

On screens ≤1024px, the `.zw-service-row--reverse` direction override is removed and all rows stack to a single column.

The page ends with a CTA banner: "¿No encuentras lo que buscas?" linking to `/licencias`.
