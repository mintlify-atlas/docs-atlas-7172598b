# Homepage

The homepage is registered at `/` and is defined by the `zonaweb_homepage` template, which replaces the `#wrap` element of `website.homepage`.

## Hero section

The hero section (`.zw-hero`) features:

- An animated label "Soluciones Tecnológicas Profesionales" with a pulsing dot indicator
- A headline: "Transformamos tu empresa con tecnología de vanguardia"
- A description paragraph covering software, networks, support, and security
- Two action buttons: **Solicitar propuesta** → `/licencias` and **Ver servicios** → `/servicios`
- Three animated statistics counters (using `data-count` attributes):

| Stat | Value |
|---|---|
| Años de experiencia | 10+ |
| Proyectos completados | 120+ |
| Clientes satisfechos | 98% |

- An image carousel with 3 slides (hero1.jpg, hero2.jpg, hero3.jpg) with dot navigation
- An ISO 27001 badge overlay

The hero visual column uses the `.animate-scale-in.delay-3` classes, while the content column uses `.animate-fade-up`. On screens up to 1024px, the visual column (`.zw-hero__visual`) is hidden.

## Security plan pricing

The homepage includes a pricing section (`.zw-plans`) with a dark navy background and two tiers. The featured card displays a "Popular" badge via the `::before` pseudo-element on `.zw-plan-card--featured`.

| Feature | Plan Estándar ($49/mo) | Plan Premium ($79/mo) |
|---|---|---|
| Antivirus protection | Basic | Advanced |
| Support | Email | 24/7 technical support |
| Updates | Monthly | Real-time |
| Monitoring | Basic | Proactive |
| Cloud backup | — | Included |
| Purchase link | `/shop` | `/shop` |

<Note>
Plan purchases go through the Odoo eCommerce shop (`/shop`), powered by the `website_sale` dependency.
</Note>

## Testimonials

A single testimonial card (`.zw-testimonial-card`) is displayed, bordered on the left with the primary blue color. The author avatar uses text initials (`JD`) rather than a photo image, rendered via `.zw-testimonial-avatar`.

## Template definition

The homepage customizes Odoo's default homepage via XPath:

```xml
<template id="zonaweb_homepage"
          key="zonaweb_website.zonaweb_homepage"
          name="Zonaweb Homepage"
          inherit_id="website.homepage">
    <xpath expr="//div[@id='wrap']" position="replace">
        <!-- full page content -->
    </xpath>
</template>
```
