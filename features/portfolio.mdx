# Portfolio page

**URL:** `/portafolio`  
**Template:** `zonaweb_website.portafolio_page`  
**Auth:** Public

The portfolio page showcases six completed projects organized by category, with client-side JavaScript filtering.

## Filter categories

| Filter button | `data-filter` value | Description |
|---|---|---|
| Todos | `all` | Show all projects |
| Desarrollo Web | `software` | Web development projects |
| Aplicaciones | `app` | Application projects |
| Infraestructura | `infra` | Infrastructure projects |
| Seguridad | `security` | Security projects |

Filtering is client-side: clicking a button removes the `active` class from all buttons, adds it to the clicked one, then sets `display: none` on cards whose `data-cat` attribute does not match the selected filter (or shows all cards when `all` is selected). See [JavaScript](/features/javascript) for the implementation.

## Card structure

Each project card (`.zw-pf-card`) contains:
- A category badge (`.zw-pf-card__cat`) positioned at the bottom-left of the image, overlaid on top of the image gradient
- A `data-cat` attribute on the card root for JavaScript filtering
- Technology tags (`.zw-tag` for grey, `.zw-tag--blue` for blue accent)
- Project metadata (`.zw-pf-card__meta`): industry (`fa-building`), duration (`fa-calendar`), team size (`fa-users`)

The full portfolio grid uses `repeat(auto-fill, minmax(340px, 1fr))`, collapsing to a single column on screens ≤768px.

The page ends with a CTA banner: "¿Listo para tu próximo proyecto?" linking to `/licencias`.
