# CSS components

The theme stylesheet (`static/src/css/zonaweb_style.css`) uses a BEM-inspired naming convention with the `zw-` prefix for all custom components.

## Naming convention

```
.zw-[component]__[element]--[modifier]
```

- `zw-` ŌĆö Zonaweb namespace prefix (prevents collision with Odoo/Bootstrap classes)
- `[component]` ŌĆö the block (e.g., `hero`, `service-card`, `plan-card`)
- `__[element]` ŌĆö a child element (e.g., `__title`, `__icon`, `__img`)
- `--[modifier]` ŌĆö a variant (e.g., `--featured`, `--reverse`, `--white`)

## Asset bundle

The CSS is loaded via the `web.assets_frontend` bundle:

```python
'assets': {
    'web.assets_frontend': [
        'zonaweb_website/static/src/css/zonaweb_style.css',
    ],
    ...
}
```

<Info>
The CSS goes in `web.assets_frontend` rather than `website.assets_frontend` because it needs to load on all frontend pages. The `website.assets_frontend` bundle is reserved for JavaScript that requires the website editor context.
</Info>

## Animation classes

| Class | Keyframe | Description |
|---|---|---|
| `.animate-fade-up` | `fadeUp` | Fades in while sliding up from 30px |
| `.animate-fade-in` | `fadeIn` | Simple opacity fade in |
| `.animate-slide-right` | `slideRight` | Slides in from -30px left |
| `.animate-scale-in` | `scaleIn` | Scales from 0.92 to 1 with fade |
| `.delay-1` through `.delay-6` | ŌĆö | Animation delay in 0.1s increments (0.1sŌĆō0.6s) |

<Tip>
The `animate-fade-up` and `animate-scale-in` classes are CSS animations that fire on page load. Scroll-triggered reveal for cards (`.zw-service-card`, `.zw-pf-card`, etc.) is handled separately by the JavaScript intersection observer. See [JavaScript](/features/javascript) for details.
</Tip>
